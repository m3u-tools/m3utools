M3U TOOLS

This site contains FREE tools to manage your m3u and m3u8 files. You can merge files or shrink one down. We also have a xml and xml.gz merger. There is NO logs NO saved info and NO sign up needed.We also do not track our users.

M3U M3U8 Editor

This tool will allow you to shrink down your m3u/m3u8. If your playlist is too large no problem load it up and only select what you want to keep. You can also merge you m3u/m3u8 together to make one list. Click the image to head over to the tool.

XML Editor

This tool will allow you to add up to 10 xml urls so you can merge them into one file. Click the image to head over to the tool.

XML.gz Editor

This tool will allow you to add up to 5 .gz file so you can merge them into one file. Click the image to head over to the tool.
